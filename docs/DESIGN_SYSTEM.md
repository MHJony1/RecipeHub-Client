# RecipeHub Design System

This document defines the visual language of the entire application.

Every page, component and layout must follow this design system.

Never create inconsistent UI.

---

# Design Philosophy

The design should feel

- Modern
- Clean
- Minimal
- Professional
- Friendly
- Consistent

Avoid visual clutter.

Whitespace is important.

---

# Color Palette

Primary

Green

- Primary Buttons
- CTA
- Active Links
- Success Elements

Secondary

Orange

Used for

- Highlights
- Badges
- Icons

Accent

- Featured Labels

White

Gray

Black

Background

Text
