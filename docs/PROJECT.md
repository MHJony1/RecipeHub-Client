# 📖 Project Overview

RecipeHub is a modern full-stack recipe sharing platform where users can discover, search, filter and explore cooking recipes from different categories.

Authenticated users can publish their own recipes and manage them from a personalized dashboard.

The application should provide a clean, responsive and production-quality user experience while demonstrating modern full stack development skills using Next.js, TypeScript, MongoDB and JWT Authentication.

This project is intended for portfolio purposes and must satisfy all assignment requirements.

# ❗ Problem Statement

People often struggle to find organized recipes with proper cooking instructions.

Many recipe websites are overloaded with advertisements, poor navigation and inconsistent layouts.

RecipeHub solves this by providing a clean, modern and easy-to-use recipe platform.
